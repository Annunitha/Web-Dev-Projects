# Nutri Plan

A personalized nutrition planning web application designed to help users create tailored diet plans based on their dietary preferences, health goals, and lifestyle. The app provides daily meal suggestions, tracks nutrient intake, and offers insights into maintaining a balanced diet.

## Features

- **User Authentication:** Create an account, set up a profile with body metrics, dietary preferences, and health goals.
- **Dashboard:** At-a-glance view of daily macros, calorie targets, water intake, day streaks, and quick stats.
- **Meal Suggestions:** AI-like mock generation of meal plans filtered by diet (Vegan, Keto, High-Protein, etc.) and calorie needs.
- **Food Logging:** Search and log food portions to track detailed macro and calorie consumption.
- **Progress Tracking:** Interactive charts tracking weight trends, daily calories, weekly macros, and water intake over time.
- **Community Feed:** Share meal plans, view trending tags, and interact with other users' nutrition journeys.
- **Admin Dashboard:** A simple admin view for monitoring total registered users and app usage statistics.
- **Premium Dark Theme:** A sleek black, gold, and green responsive user interface utilizing CSS grid and flexbox.
